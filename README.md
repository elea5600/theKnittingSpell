# the Knitting Spell 💕🧶


An application that helps to calculate stitches and rows for a knitting project!
Especially helpful for calculating decreases and increases.

Enjoy !🧚

## SooN
Will be hosted on a cloud ✨⛅

## Notes:
this was a very quick late night programming, as I couldnt find my calculator and also didnt wanna abuse AIs.. 
So its not the prettiest code, but it does what it should!
If you have any notes, complaints or other input, feel free to let me know!

## Run with Docker

Build the image:

```bash
docker build -t knithelper .
```

Run the container:

```bash
docker run --rm -p 8080:8080 knithelper
```
